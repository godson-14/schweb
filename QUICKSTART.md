# Quick Start Guide

## To Run Locally

1. Open PowerShell in the project folder
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
4. Open browser to: `http://localhost:3000`

## To Upload & Deploy

Follow the steps in **DEPLOY.md**
